# parallelworks/mpi-workflows

This repository is a collection of MPI workflows for
demonstrating the multi-node usage of clusters connected
to the Parallel Works ACTIVATE platform.

In general, the code in `run_on_cluster` is meant to be
run manually on a cluster (i.e. via a terminal) and over 
time workflow wrappers can be added to this 
*work-in-progress* repository.
